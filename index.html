Mes projets

Dans ce projet, plusieurs planches animées s'assemble pour créer le générique d'un film intitulé "The Scream of Joy". L'histoire déprimante d'un homme qui devient fou après qu'un clown tue sa femme. Bien sûr, ce film n'existe pas!

Cette cymbale a été modélisé en 3D pour être incluse dans un site Web pour le groupe de musique Set It Off. Quand l'utilisateur défile dans le site, la cymbale le suit et change d'animation quand il monte, descend ou reste immobile.

Pétale est une vidéo d'art ou tout est possible. Y-aurait-il un message caché derrière ce projet? Où se passe les scènes? Pourquoi des pétales? Tant de questions qui on peu d'importance pour apprécier cette vidéo d'art. J'étais le directeur de la photographie pour ce projet.

Victoria est un court-métrage montrant la vie d'une fille du même nom. C'est une artiste en manque d'inspiration qui s'aventure pour en trouver, que ce soit un poisson ou une banane, toutes les inspirations sont valables pour elle! J'étais le directeur de la photographie pour ce projet.
